### ✨ Tính năng nổi bật (Key Features)
- ✅ **Bán hàng thông minh:** Giao diện Dark Mode mượt mà, hỗ trợ tách/gộp hóa đơn.
- ✅ **Thanh toán VietQR:** Tự động tạo mã QR động kèm số tiền và nội dung chuyển khoản.
- ✅ **Xử lý bất đồng bộ (@Async):** Gửi Email hóa đơn tự động qua Google SMTP mà không gây trễ hệ thống.
- ✅ **Báo cáo trực quan:** Theo dõi doanh thu 7 ngày qua biểu đồ Chart.js thời gian thực.

### 🛠️ Công nghệ sử dụng (Tech Stack)
- **Backend:** Java 17, Spring Boot 3.x, Spring Data JPA.
- **Frontend:** Thymeleaf, Bootstrap 5, JavaScript (ES6+).
- **Cloud & DevOps:** Render (App Hosting), Aiven (Managed MySQL), GitHub Actions.
